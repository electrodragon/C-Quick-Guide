Functions Description (LIB: ctype.h)
isalpha() checks character is alphabet
isdigit() checks character is digit
isalnum() Checks character is alphanumeric
isspace() Checks character is space
islower() Checks character is lower case
isupper() Checks character is UPPER CASE
isxdigit() Checks character is hexadecimal
iscntrl() Checks character is a control character
isprint() Checks character is a printable character
ispunct() Checks character is a punctuation
isgraph() Checks character is a graphical character
tolower() Converts character to lower case
toupper() Converts character to UPPER CASE
